# Loop Habit Tracker Lite (Node.js)

A lightweight habit tracker inspired by Loop Habit Tracker.

## Features

- Add and delete habits
- Mark habit as done for today (or undo)
- Automatic streak calculation based on consecutive days
- Data persisted to `data/habits.json`

## Run locally

```bash
npm install
npm start
```

Open http://localhost:3000

## Test

```bash
npm test
```
